## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## debug the app
```bash
# debug with vscode
run F5, more detail with ./vscode/launch.json

# debug with chrome
$ npm run start:debug
open chrome with  chrome://inspect/ 
```
